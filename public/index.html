Event to WhatsApp POC

Capture a community event, save it in the backend, then forward it to WhatsApp.

This proof of concept keeps the flow simple: the browser submits the form to FastAPI, FastAPI stores the event in SQLite, and then it tries to forward the same event to a WhatsApp sender service.
